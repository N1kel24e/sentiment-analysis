## Sentiment Analysis

A machine learning model that determines whether a movie review is positive or negative.
Accuracy: 89.43%

## Dataset
IMDB Dataset of 50K Movie Reviews
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

## Technologies
- Python
- pandas
- scikit-learn
- BeautifulSoup
